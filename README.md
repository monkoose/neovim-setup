# neovim-setup

1. Clone the repository

    ```
    git clone https://github.com/monkoose/neovim-setup
    ```

2. Run installer.sh

    ```
    sh ./installer.sh
    ```

3. Open neovim and install plugins

    ```vim
    :PlugInstall
    ```
4. Install CoC plugings
    ```vim
    :CocInstall <coc-plugin>
    ```

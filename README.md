# neovim-setup

1. Download installer

    ```
    curl -Lo installer.sh https://raw.githubusercontent.com/monkoose/neovim-setup/master/installer.sh
    ```

2. Run installer.sh

    ```
    sh ./installer.sh
    ```

3. Open neovim and install plugins

    ```vim
    :PlugInstall
    ```
4. Install CoC plugings
    ```vim
    :CocInstall <coc-plugin>
    ```
